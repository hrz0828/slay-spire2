# Site Configuration Notes

- Production domain: `https://sts2hub.com/`
- Baidu submit site parameter: `https://sts2hub.com`
- Baidu submit token: configured in `submit-to-baidu.js`
- Chinese card URLs submitted to Baidu use canonical no-prefix paths such as `https://sts2hub.com/cards/Bash`.
- Baidu submit failures are logged but should not fail the site build.
- Do not change the production domain or Baidu token unless the site owner explicitly requests it.
