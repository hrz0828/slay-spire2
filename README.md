# Slay the Spire 2 Guide

A bilingual Slay the Spire 2 guide and tools site built with Next.js App Router, Tailwind CSS, and lightweight JSON dictionaries.

## Routes

- `/zh` Chinese site
- `/en` English site

## Development

```bash
npm install
npm run dev
```

## Cloudflare

Cloudflare now recommends the OpenNext adapter for Next.js deployments.

```bash
npm run preview
npm run deploy
```

Set `NEXT_PUBLIC_SITE_URL` to the production domain for canonical URLs and sitemaps.
